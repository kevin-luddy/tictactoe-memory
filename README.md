# TicTacToe Memory Challenge

A fun tic-tac-toe game with a memory twist - the pieces become invisible when the countdown ends!

## Quick Start

```bash
python play.py
```

That's it! The game will start automatically in your browser.

## How It Works

1. Play tic-tac-toe against the computer
2. A countdown timer shows during your turn
3. When the timer ends, all pieces become **invisible**!
4. You must remember where pieces are to make smart moves
5. Pieces reappear after you make your move

## Difficulty Levels

- **Easy** - 10 seconds to memorize
- **Medium** - 5 seconds to memorize
- **Hard** - 1 second to memorize (for memory champions!)

## Features

- Kid-friendly sounds and colorful design
- Win streak counter to track your progress
- Three difficulty levels
- Fun visual effects and animations

## Requirements

- Python 3.8+
- FastAPI
- Uvicorn

Install dependencies:
```bash
pip install fastapi uvicorn
```
